# ARP 欺骗安全实验

## 项目简介
本项目围绕 TCP/IP 协议下的网络攻击展开，重点研究 ARP Spoofing（ARP 欺骗）与 IP Spoofing（IP 伪造）攻击的原理、实验过程及防御策略。

## 项目目标
- 理解 TCP/IP 协议结构
- 分析 ARP 欺骗攻击原理
- 研究 IP 地址伪造攻击过程
- 了解企业网络中的安全防御措施

## 实验环境
- Kali Linux
- Local Area Network (LAN)
- arpspoof tool
- Packet monitoring tools

## 项目内容
1. TCP/IP 协议基础说明
2. ARP 攻击实验
3. IP Spoofing 原理分析
4. 网络攻击防御策略总结

## 技术关键词

- TCP/IP
- ARP Spoofing
- IP Spoofing
- Network Defense
